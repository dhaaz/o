Baiklah, izinkan aku memberi tahu kamu apa yang tidak aku lakukan: Aku tidak menghosting file website ini secara ilegal. Dan aku pun tidak menyimpan konten yang dilindungi hak cipta di situs web ini. Semua file yang ada di website ini hanya untuk memperkenalkan tujuanku mendorong semua pengguna untuk membeli CD atau DVD dari film atau musik yang disukainya.

Ketika kamu menjelajahi halaman series, 99% ada pemutar videonya. Yang perlu kamu lakukan hanyalah mengklik Tombol Putar, itu saja. Jika tidak berhasil (jangan menjudge!
